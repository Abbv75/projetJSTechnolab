================================INSECT GAME======================================

INSECT GAME EST UN JEU DE SELECTION D'INSECT.
LE JEU SE DEROULE COMMME SUITE:
1-PAGE ACCEUILL : CLIQUER SUR LE BOUTTON JOUER POUR LANCER LE JEU
2-PAGE DE SELECTION : SELECTIONNER VOTRE INSECTE PREFERE
3-PAGE DE JEU : SUR CETTE PAGE VOUS AUREZ UN TIMER QUI S'INCREMENTERA DE 0 A 20 ET UN SCORERE
LE JEU SE DEROULERA SUR CETTE PAGE LE PRINCIPE EST SIMPLE DURANT LE JEU L'UTLISATEUR DEVRA CLIQUE SUR 20 INSECTE EN 10 SECONDE (LE JEU SARRETERA ET DEMANDERA SI VOUS VOULEZ REJOUEZ) POUR GAGNER
SI ON LE TIMER ET LE SCORER CONTINUERONT DE S'INCREMENTER.

TECHNLOGIES UTILISES : HTML/CSS/JS
